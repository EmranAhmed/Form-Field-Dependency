Form-Field-Dependency

Simple Form Field Dependency. Conditional Form Fields.

Download .zip Download .tar.gz View on GitHub

Include jQuery

<script type="text/javascript" src="http://code.jquery.com/jquery-1.11.3.min.js"> </script>

Include Form-Field-Dependency

<script type="text/javascript" src="form-field-dependency.js"> </script>

Create your form components

<div class="form-group">
    <label for="InputEmail">Email address:</label>
    <input type="text" class="form-control" id="InputEmail" placeholder="Email">
</div>

Activate plugin

jQuery(function ($) {
    $('[data-depends]').formFieldDependency();
});

Add a simple css

[data-depends]{
    display:none
}

Add your global field rule.

You can write your field rule globally or on element attribute. Field rule should written in JSON data.

Code: Globally defined like

$.fn.formFieldDependency({
'rules': {
     '#only-email' : {
        '#InputEmail' : {'type' : 'regexp', 'pattern' : '[a-z]+@[a-z]+.[a-z]', 'modifier' : 'i'}
     },
     '#only-equal' : {
        '#InputEmail' : {'type' : 'equal', 'value' : ['lorem', 'ipsum']}
      }
    }
});

Code: On attribute 

$('[data-depends]').formFieldDependency({
'attribute' : 'data-depends', // Attribute name where you write your roles
'rules'     : {
    'DependencySelector' : {  // like #id, .class, input[name]
        'DependsOnSelector' : {'type' : 'regexp', 'pattern' : '[a-z]+@[a-z]+.[a-z]', 'modifier' : 'i'}
    },
    'DependencySelector' :
    {
        'DependsOnSelector' : {'type' : 'equal', 'value' : ['lorem', 'ipsum']}
    }
}
});

Options

  1. attribute write rules on field attribute. You can write your own html attribute name where you define your rules for each field. Default is data-depends
  2. rules global rules. You can write global rules.

Available Dependency types

empty

Show dependency field if dependsOn field value is empty.

empty type does not require any others attributes.

On attribute
data-depends="[{'#DependsOnSelector' : {'type':'empty'}}]"
On rules
'rules' : {
    '#DependencySelector' : {
        '#DependsOnSelector' : {'type' : 'empty'}
    }
}

DependsOnSelector should be a input, checkbox, radio or select box

Example Output:
Example Code:
<div class="form-group">
    <label for="exampleInput1">Text Box</label>
    <input type="email" name="checkEmpty" class="form-control" id="exampleInput1" placeholder="">
</div>

<div class="form-group" data-depends="[{'input[name=checkEmpty]':{'type':'empty'}}]">
    <label for="exampleInput2">This will hide if you write something on Text Box</label>
    <input type="email" class="form-control" id="exampleInput2" placeholder="">
</div>

!emptynotemptynot-empty

Show Dependency field if DependsOn field value is not empty.

There is many alias of not empty type: !empty, notempty, not-empty, notEmpty.

!empty type does not require any others attributes.

On attribute
data-depends="[{'#DependsOnSelector' : {'type':'empty'}}]"
On rules
'rules' : {
    '#DependencySelector' : {
        '#DependsOnSelector' : {'type' : '!empty'}
    }
}

DependsOnSelector should be a input, checkbox, radio or select box

Example Output:

equal == =

Show dependency field if dependsOn field value is equal to given value.

There is many alias of equal type: equal, ==, =.

Define value in array or in string.

Checking type strict true to check strictly equal or just match anyone in array.

Default strict value is false

Note: strict value true will effective if you use it on multiple select or checkbox.

Code: on attribute. 
data-depends="[{'#DependsOnField' : {'type':'equal', 'value':'lorem'}}]"
data-depends="[{'#DependsOnField' : {'type':'equal', 'value':['lorem', 'ipsum']}}]"
data-depends="[{'#DependsOnField' : {'type':'equal', 'value':['lorem', 'ipsum'], 'strict':true}}]"
