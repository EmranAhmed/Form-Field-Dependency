Form-Field-Dependency

OnFly Simple Form Field Dependency. Conditional Form Fields.

Download .zip Download .tar.gz View on GitHub

Include jQuery

<script type="text/javascript" src="http://code.jquery.com/jquery-1.11.3.min.js"> </script>

Include Form-Field-Dependency

<script type="text/javascript" src="form-field-dependency.js"> </script>

Create your form components

<div class="form-group">
    <label for="InputEmail">Email address:</label>
    <input type="text" class="form-control" id="InputEmail" placeholder="Email">
</div>

Activate plugin

jQuery(function ($) {
    $('[data-depends]').formFieldDependency();
});

Add a simple css

[data-depends]{
    display:none
}

Add your field rule.

You can write your field rule globally or on fly. Field rule should written by JSON data.

Code: Globally defined like

$.fn.formFieldDependency({
'rules': {
     '#only-email' : {
        '#InputEmail' : {'type' : 'regexp', 'pattern' : '[a-z]+@[a-z]+.[a-z]', 'modifier' : 'i'}
     },
     '#only-equal' : {
        '#InputEmail' : {'type' : 'equal', 'value' : ['lorem', 'ipsum']}
      }
    }
});

Code: On Fly 

$('[data-depends]').formFieldDependency({
'attribute' : 'data-depends',
'rules'     : {
    '#only-email' : {
        '#InputEmail' : {'type' : 'regexp', 'pattern' : '[a-z]+@[a-z]+.[a-z]', 'modifier' : 'i'}
    },
    '#only-equal' :
    {
        '#InputEmail' : {'type' : 'equal', 'value' : ['lorem', 'ipsum']}
    }
}
});

Options

  1. attribute on fly rules field attribute. You can write your own html attribute name where you define your on-fly rule for each field. Default is data-depends
  2. rules global rules. You can write global rules.

Available Dependency types

{
   '#exampleInputEmail3' : {'type':'empty'},
   '#exampleInputEmail3' : {'type':'!empty'},
   '#exampleInputEmail3' : {'type':'equal', 'value':'b'},
   '#exampleInputEmail3' : {'type':'equal', 'value':['a', 'b', 'c']},
   '#exampleInputEmail3' : {'type':'equal', 'value':['a', 'b', 'c'], 'strict':true},
   '#exampleInputEmail3' : {'type':'equal', 'like':'#exampleInputEmail4', 'strict':true, 'empty':true},
   '#exampleInputEmail3' : {'type':'!equal', 'value':'a'},
   '#exampleInputEmail3' : {'type':'!equal', 'value':['a', 'b', 'c']},
   '#exampleInputEmail3' : {'type':'regexp', 'pattern':'[a-z]+@[a-z]+.[a-z]+', 'modifier':'gi'},
   '#exampleInputEmail3' : {'type':'compare', 'value':'100', 'sign':'>'}, // > or < or >= or <=
   '#exampleInputEmail3' : {'type':'length', 'sign':'>', 'value':'100'},
   '#exampleInputEmail3' : {'type':'range', 'value':['10', '50']},
   '#exampleInputEmail3' : {'type':'range', 'min':'10', 'max':'50']},
}

Empty

empty

Show dependency field if dependsOn field value is empty. Does not require any other attribute.

Code: on attribute
data-depends="{'#DependsOnField' : {'type':'empty'}}"
Code: on rules
'rules' : {
            '#DependencyField' : {
                '#DependsOnField' : {'type' : 'empty'}
            }
           }

Example Output:
Example Code:
<div class="form-group">
    <label for="exampleInput1">Text Box</label>
    <input type="email" name="checkEmpty" class="form-control" id="exampleInput1" placeholder="">
</div>

<div class="form-group" data-depends="{'input[name=checkEmpty]':{'type':'empty'}}">
    <label for="exampleInput2">This will hide if you write something on Text Box</label>
    <input type="email" class="form-control" id="exampleInput2" placeholder="">
</div>


Equal

equal == =

Show dependency field if dependsOn field value is equal to given value. Define value on array or on string. Checking type strict to check strictly equal or just match anyone in array.

Code: on attribute. 
data-depends="{'#DependsOnField' : {'type':'equal', 'value':'lorem'}}"
data-depends="{'#DependsOnField' : {'type':'equal', 'value':['lorem', 'ipsum']}}"
data-depends="{'#DependsOnField' : {'type':'equal', 'value':['lorem', 'ipsum'], 'strict':true}}"

Example Output:
Example Code:
<div class="form-group">
    <label for="exampleInput5">A box will show if you write "lorem" or "ipsum".</label>
    <input type="email" name="" class="form-control" id="exampleInput5" placeholder="">
</div>

<div class="form-group" data-depends="{'#exampleInput5':{'type':'equal', 'value':['lorem','ipsum']}}">
    <label for="exampleInput6">Hidden Field</label>
    <input type="email" class="form-control" id="exampleInput6" placeholder="">
</div>


Not Empty

!empty notempty not-empty notEmpty

Show dependency field if dependsOn field value is not empty. Does not require any other attribute.

Code: on attribute
data-depends="{'#DependsOnField' : {'type':'!empty'}}"
Code: on rules
'rules' : {
            '#DependencyField' : {
                '#DependsOnField' : {'type' : '!empty'}
            }
           }

Example Output:
Example Code:
<div class="form-group">
    <label for="exampleInput3">A box will show if you write something.</label>
    <input type="email" name="checkNotEmpty" class="form-control" id="exampleInput3" placeholder="">
</div>

<div class="form-group" data-depends="{'#exampleInput3':{'type':'!empty'}}">
    <label for="exampleInput4">Hidden Field</label>
    <input type="email" class="form-control" id="exampleInput4" placeholder="">
</div>
